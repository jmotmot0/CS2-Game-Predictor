# MVP scope

Первая версия проекта предсказывает победителя профессионального матча CS2
по предматчевым командным признакам.

На первом этапе реализуются:
- сбор исторических матчей;
- базовая очистка данных;
- единая схема матчевых данных;
- baseline на Elo;
- первичный data audit.

На этом этапе НЕ реализуются:
- сложный player-level feature engineering;
- полный roster tracking;
- продвинутый map pool analysis;
- сложные ансамбли моделей.

Цель этапа:
получить воспроизводимый pipeline от сырых матчей до baseline-предсказания.
