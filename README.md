# CS2 Match Outcome Prediction

Учебный проект: предсказание исходов профессиональных матчей CS2
по предматчевым признакам.

## Цель
Построить baseline-модель, которая оценивает вероятность победы команды
до начала матча.

## MVP
Первая версия включает:
- загрузку исторических матчей;
- очистку и нормализацию данных;
- базовый анализ данных;
- Elo-baseline.

## Структура проекта
- `data/raw` — сырые данные
- `data/interim` — промежуточные таблицы
- `data/processed` — готовые датасеты для моделирования
- `notebooks` — исследовательские ноутбуки
- `src` — код проекта
- `reports` — графики, результаты, заметки
